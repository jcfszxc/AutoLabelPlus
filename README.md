# AutoLabelPlus

AutoLabelPlus 是一个基于 LabelImg 开发的升级版自动标注工具，旨在提供更高效的图像标注解决方案。

## 功能特点

- 保留 LabelImg 的所有基础功能
- 集成自动标注功能，提高标注效率
- 友好的用户界面
- 支持多种图像格式
- 图片缩略图预览功能
- 标注目标框缩略图快速定位
- 支持通过文件列表快速切换图片

## 安装要求

- Python 3.8+
- PyQt5
- 其他依赖项 (详见 requirements.txt)

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/jcfszxc/AutoLabelPlus.git
```

2. 进入项目目录
```bash
cd AutoLabelPlus
```

3. 运行程序
```bash
python main.py
```
